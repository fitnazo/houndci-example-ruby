# Start RSpec
Learn Rspec Testing for ruby step by steps

#Installation 
  * Clone the repo to your system and `bundle` it.
  * Add your `class` file inside the lib folder.
  * Add your `spec` file inside the spec folder.
  * run command `rspec`
